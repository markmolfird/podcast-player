Podcast Player Prime :: About

Podcast Player Prime is a lite and easy-to-use app, designed to play and download audio podcasts. It has a built-in search tool that uses the iTunes database to search for specific keywords. Moreover, you can add a podcast feed by a feed URL. Also, there is a - topchart - section which you can use to add a few pre-defined and popular podcast episodes. Each of these features is explained in the following.

In Podcast Player Prime, you can adjust several settings, including the app theme color. There is a handy color palette in the settings tab which you can use to select different colors for various parts of the app. Moreover, there are some general settings to change the player's behavior. For managing downloaded items, including image, audio, etc., there is a CACHE section to view or delete a cached item(s). Please read below to get more info about each section in this app. You can also visit app's homepage to get detailed instruction of how to work with Podcast Player Prime.

Please note, Podcast Player Prime is still in beta and may not work properly for all podcasts (various audio formats). Plus, you may see some bugs in different functions (i.e. player, color palette, cache, etc.). This project is still on active development; in future versions, support for video podcasts will be added. If you see any bugs or have a feature request in your mind or other suggestions for this app, please use the bug report form to report bugs or other suggestions.

Podcast Player Prime :: How to work

Podcast Player Prime has five main tabs (pages) plus the current tab which you can open from the buttons in the top toolbar. At the top toolbar, there are five main buttons and one reload button; from right to left, there is a reload button to reload the app UI. Next, there are settings, manage, add, player, and refresh tabs. The refresh button is for reloading all podcast data (including feeds & images)

Podcast Player Prime :: Front tab (homepage)

The front tab or Homepage is where you can see all your subscribed podcasts. If there is no subscribed podcast (i.e. when you first install the app), this help section is shown instead. To start adding a podcast, please follow these 4 steps.

1. Please press on the PLUS sign (+) button from the top banner to open the add-new-podcast page (to search for the new item(s)).

2. In the new page, click on the - Search - tab (or any other tabs, depending on your need (i.e. Top Chart, Add Feed or Subscribed podcasts).

3. In the search Tab, type-in your keyword (i.e. ted, radio, etc.) and wait to see the results (if there is any). If you see the search button is spinning, the app is fetching new data from the iTunes server. Once the results are shown, hit the small - mark - icon next to each podcast to add it to the subscribed list. The item will be saved and listed in the - Subscribed - podcasts tab. Once you added all your desired items to the subscribed list, please press on the home button from the top toolbar to open the front page.

4. As mentioned above, pressing on the home button will open the front page with all your subscribed items listed. Please click on an item on the front page to open it on the player page. Within the player page, you will see the player toolbar and all episodes for the current podcast. Each episode has 3 buttons. From the left, there is a play button. On the right, there are add-to-playlist (top) and mark-as-seen (bottom) buttons. The main player at the top has 5 main and, an extra 4 buttons (click on the down-arrow to see the extra buttons). These buttons are self-explanatory as they are standard in all medial players.

5. The big podcast image is shown at the top empty area on the player page. You can also see the current and total time next to the home button. Clicking on the home button directs you to the front page, where you can see all the subscribed podcasts. To edit the list, please hit the plus button again. If you want to view/manage the cache or adjust other settings, please read the following sections.

6. By default, Podcast Player Prime fetches maximum 15 episodes for each podcast. If you want to increase this number, please go to the settings page and adjust this max value based on your needs. Please note, increasing the number to the values greater than 100, may slow down the app's performance.

Podcast Player Prime :: Settings tab

Within the settings tab, there are 4 main buttons. The first one is for adjusting the theme color. You can change the color for all the elements in the app. If you click on this button, a color-palette appears. With this color palette, you can change the color for the fonts, buttons, and background color. Simply click on the - font - button and choose your desired color from the palette. Next, click on the - button - and choose the desired color for all buttons. Similarly, choose a color for the - background - color. All these colors are saved to memory. There is also a slider that can be used for adjusting the color for all unselected buttons as-well-as some title areas. Other than the color palette, you can choose a few pre-defined themes placed at the bottom of the theme color page. At the moment, there are 5 themes to choose from.

The next tab on this page is the general settings where you can adjust several options according to your needs. The third tab is the cache settings where it shows the cached items for all the podcasts, images, audio and video items. Please note, deleting a cache item, completely removes it from the disk and this operating is irreversible. There is also a big trash button at the top, where you can use to delete all cached items in each category (podcast, image, audio or video). The last tab is for the Kernel logs (may be used for detecting internal errors or fixing app's bugs). While performing any operation within the app, you can visit this tab to get more info about what is being done behind-the-scenes. Once you are done with the settings page, press on the home button to open the front page.

Podcast Player Prime :: Manage tab

The button next to the settings is the manage button. Click on the manage button to see your downloaded tracks, playlist items, and marked-as-seen tracks. Next to each item, there is a remove button that you can use to delete the item from the corresponding list.

Podcast Player Prime :: Player tab

Once you go the front page, the fifth button from the top toolbar opens the player page. In the beginning when there is no subscribed podcast, this page in empty. But if you add a podcast to the subscribed list, you will see the subscribed podcast and all the episodes on the player page. Within the player page, there are two sections. The top section shows the main player buttons and podcast image.

The main player has a big play button. Once pressed, it will play the last episode. Please note, you need to first select an episode from the bottom list and then use this big play button to play or pause any item, otherwise, it shows you an error. There are also forward and backward buttons as well as fast-forward and fast-background buttons within the main player section.

At the bottom, all the episodes are listed for the current podcast (in card-view). Each episode item has 3 buttons. From the left, there is a play button that you can use to play the podcast. On the top-right there is an add-to-playlist button, to add the current track to the playlist. The last button is for mark-as-read, if pressed, the item will be marked as read and removed from the current list. You can show all the mark-as-seen items by pressing on the - eye - icon within the main player section (click on the small down-arrow in the main player to see the eye icon and all the other hidden icons).

Podcast Player Prime :: Refresh Button

Use the refresh button to reload data and images for all your subscribed podcasts. Please note, this button only updates the data related to the subscribed podcasts, and has no effect on other podcasts such as the searched items or the top chart list. To update data for a specific podcast, please head to the front page (press home button) and then press the refresh button at the bottom-right corner for each podcast item. Wait for few seconds for the podcast data to be updated.
